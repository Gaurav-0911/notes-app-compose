## How to Run
1. Open the project in Android Studio.
2. Allow Gradle sync to complete.
3. Connect a physical device or start an emulator.
4. Click the Run button to launch the app.

## How to Extend
- Room Database can be integrated to persist notes permanently.
- Delete functionality can be added to remove notes.
- Search feature can be implemented to filter notes.
