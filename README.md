# Notes App – Task 1

## Overview
This is a simple Notes application developed using Kotlin and Jetpack Compose as part of Android App Development – Task 1.

The app allows users to add notes and view them in a list using a modern Android UI.

---

## Features
- Single Activity application
- Built with Jetpack Compose
- Add notes using a dialog
- Display notes using LazyColumn
- In-memory data storage using ViewModel

---

## Tech Stack
- Kotlin
- Jetpack Compose
- Android ViewModel
- Material 3

---

## How to Run the App
1. Open the project in Android Studio.
2. Allow Gradle sync to complete.
3. Connect a physical device or emulator.
4. Click the Run button to launch the app.
